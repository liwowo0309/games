# Three Kingdoms Duel (简单双人版)

小朋友也能玩的双人三国志小品。离线运行，直接打开 `index.html` 即可。

## 快速玩法
- 两人轮流操作同一台设备。
- 回合开始拿金币：基础金币 + 城市数。
- 点自己的城市操作。
- 招兵：步兵3金、骑兵5金、弓兵4金。
- 移动：点“选出兵城市”，再点相邻城市。如果是敌城会自动战斗。
- 占领对手都城获胜（魏：许昌，蜀：成都）。

## 战斗（更简单）
- 只看人数：谁多谁赢，同数守方赢。
- 赢的一方会失去跟对手人数一样多的士兵。

## 文件
- `index.html` – 界面。
- `style.css` – 样式。
- `main.js` – 规则与地图。
